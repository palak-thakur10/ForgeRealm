# ForgeRealm

ForgeRealm is a mini strategy game management system built with ASP.NET Core MVC.

## Key Features
- Player Management
- Building System
- Resource Management
- Dashboard with game-like UI
- Unit Tests for core game logic
  
## Tech Stack
- Backend: ASP.NET Core MVC
- Database: Entity Framework Core
- Frontend: Razor Pages, Bootstrap
- Testing: xUnit
